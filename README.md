# Unit Converter

A simple and efficient Unit Converter application built with Python and Tkinter.

## Features

- **Real-time Conversion**: Instantly converts values as you type.
- **Bidirectional Conversion**: Edit either side of the conversion to update the other.
- **Supported Conversions**:
  - Feet ↔ Meters
  - Pounds ↔ Kilograms
  - Pounds ↔ Grams
  - Ounces ↔ Grams
  - Inches ↔ Millimeters
  - Fahrenheit ↔ Celsius
  - Degrees ↔ Radians
  - PSI ↔ Bar

## Installation

### Running from Source
1.  Ensure you have Python installed.
2.  Clone the repository:
    ```bash
    git clone https://github.com/UncleChocolate/Unit-Converter.git
    ```
3.  Run the application:
    ```bash
    python unit_converter.pyw
    ```
    *Note: `unit_converter.pyw` is a standalone file and can be shared/run independently.*

### Running the Executable
1.  Download `UnitConverter.exe` from the repository.
2.  Double-click to run. No Python installation required.

## Requirements

- Python 3.x (for source)
- Tkinter (usually included with Python)
